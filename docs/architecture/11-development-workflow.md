# 11. Development Workflow

## Local Development Setup
1.  **Prerequisites:** Node.js >= 18.x, npm/yarn/pnpm
2.  **Initial Setup:**
    ```bash
    npm install
    cp .env.example .env
    # Fill in .env with database and auth credentials
    ```
3.  **Development Commands:**
    ```bash
    # Start all services
    npm run dev

    # Run tests
    npm run test
    ```
